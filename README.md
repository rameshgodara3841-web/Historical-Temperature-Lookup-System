# 🌡️ Historical Temperature Analysis System

## 📌 Overview
This project is a Flask-based web application that allows users to explore historical maximum temperature data for different cities and dates. It provides an easy-to-use interface for viewing past weather records.

## 🚀 Features
- Select City
- Select Date
- View Historical Maximum Temperature
- Responsive UI
- Flask Backend
- Machine Learning Dataset Integration

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- Pandas
- Scikit-learn

## 📂 Dataset
Historical weather dataset (2000–2024)

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
